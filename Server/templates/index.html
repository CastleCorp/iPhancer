REST API:

/upload: Upload image to server.

  • Methods: POST
  • Headers: form-data (image: file, resolution: text, use_gpu: text)
  • Returns: "Upload finished: image.jpg"
  • Example: /upload

/process: Run image processor.

  • Methods: GET
  • Headers: URL args (resolution: text, use_gpu: text, filename: text, process_name: text)
  • Returns: "Started new process for: filename"
  • Example: /process?resolution=orig&use_gpu=true&filename=0.jpg

/getProcessStatus: Check if the image processor is running.

  • Methods: GET
  • Headers: None
  • Returns: process_status (running or finished)
  • Example: /getProcessStatus

/getProcessedPhoto: Get image from server.

  • Methods: GET
  • Headers: URL args (filename)
  • Returns: Image file specified by filename in /enhanced
  • Example: /getProcessedPhoto?filename=0_processed.png
  • Notes: File names will always be [name]_original.png, [name]_processed.png, [name]_before_after.png
